# Python
用py做了一些小工具
## 命令
+ 查看python版本  
`py -3 -V`  
+ 运行python  
`py -3 fullpath.py`
+ .bat 运行 .py  
```
@echo off
cmd /k py -3 F:\Learn\Python\py\CheckConfig.py
```
+ 安装包,镜像  
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple chardet
+ 更新pip  
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --upgrade pip
